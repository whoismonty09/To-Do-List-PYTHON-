# To-Do List Application (Python)

## Description
This project is a simple command-line To-Do List application built using Python. It allows users to add tasks, view all tasks, and delete tasks through a menu-driven interface. This project helps in practicing basic Python concepts such as lists, loops, functions, and user input handling.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Menu-driven interface
- Easy to use command-line program

## Technologies Used
- Python
